4. CONTENT AUTOMATION SYSTEM (GITHUB ACTIONS + CLAUDE AI)
════════════════════════════════════════════════════════════════════════════════

This is the core operational advantage of Test Your Mind. Instead of manually
writing tests, you simply enter a topic into GitHub — and the entire test is
automatically created and published to the live website within minutes.

────────────────────────────────────────────────────────────────────────────────
4-1. Full Automation Pipeline
────────────────────────────────────────────────────────────────────────────────

  Step 1 — Topic Input
    You enter a test topic (e.g. "Attachment Style") into GitHub Actions
    Tool: GitHub Actions (manual trigger via web UI)

  Step 2 — AI Content Generation
    Claude AI generates: test title, intro text, 10 questions,
    4 result types with full descriptions
    Tool: Anthropic Claude API

  Step 3 — File Creation
    Generated content is saved as a structured JSON/Markdown file
    Tool: GitHub Actions

  Step 4 — Page Build
    Static site builder converts the file into a live web page automatically
    Tool: Next.js / Astro + Vercel

  Step 5 — Deployment
    New test page goes live on the website — no manual work required
    Tool: Vercel / GitHub Pages

  Step 6 — Homepage Update
    Homepage test list automatically updates to show the new test
    Tool: Automatic via data layer

────────────────────────────────────────────────────────────────────────────────
4-2. How to Trigger a New Test (Step by Step)
────────────────────────────────────────────────────────────────────────────────

  1. Go to your GitHub repository
  2. Click the "Actions" tab at the top
  3. Select the workflow: "Generate New Psychology Test"
  4. Click "Run workflow"
  5. Fill in the input: TEST_TOPIC (e.g. "What is your conflict resolution style?")
  6. Click the green "Run workflow" button
  7. Wait 1-3 minutes — your new test is live!

────────────────────────────────────────────────────────────────────────────────
4-3. GitHub Actions Workflow File
────────────────────────────────────────────────────────────────────────────────

  File: .github/workflows/generate-test.yml

  ┌─────────────────────────────────────────────────────────────────────┐
  │ name: Generate New Psychology Test                                  │
  │ on:                                                                 │
  │   workflow_dispatch:                                                │
  │     inputs:                                                         │
  │       test_topic:                                                   │
  │         description: 'Enter the psychology test topic'             │
  │         required: true                                              │
  │         type: string                                                │
  │                                                                     │
  │ jobs:                                                               │
  │   generate:                                                         │
  │     runs-on: ubuntu-latest                                          │
  │     steps:                                                          │
  │       - uses: actions/checkout@v3                                   │
  │       - name: Call Claude API to generate test content             │
  │         run: node scripts/generate-test.js                         │
  │         env:                                                        │
  │           ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}      │
  │           TEST_TOPIC: ${{ github.event.inputs.test_topic }}        │
  │       - name: Commit and push new test file                        │
  │         run: |                                                      │
  │           git config user.name 'GitHub Actions'                    │
  │           git add .                                                 │
  │           git commit -m 'Add new test: ${{ github.event.inputs.test_topic }}' │
  │           git push                                                  │
  └─────────────────────────────────────────────────────────────────────┘
